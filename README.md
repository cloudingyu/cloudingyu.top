# Hexa Realm
一个基于二维与三维间视错觉的解谜游戏. 

## 原理
通过空间视错觉, 玩家可以将二维中由三个**菱形**组成的正六边形, 幻视成三维中的一个**立方体**. 

通过这一视错觉, 玩家可以通过**旋转**二维状态下的正六边形, 从而实现三维状态下立方体的**填补**或**移除**, 最终完成一系列解密任务.
